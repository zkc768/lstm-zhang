# "Done" is defined objectively; the craft playbook is advisory, not a stop gate

**Status:** accepted (2026-06-28)

The original STOP criterion required "exemplar moves present," and Pass B checked
"exemplar-conformance" against a playbook built only from DLinear (Zeng 2023) and Grinsztajn
(2022) — terse ML craft papers, not the ICAIF/ACM-sigconf finance venue we submit to. Half the
criterion was subjective ("moves present") and the concrete half pointed at the wrong genre
(DLinear's terseness/length would push us away from venue norms). No real ICAIF exemplar had
been obtained (web outage), so venue-fit had no actual target.

**Decision:** (1) The exemplar playbook is **advisory for cross-genre structure/logic moves
only** (close-every-paragraph, goal-problem-solution rhythm, deflationary claim phrasing);
DLinear's terseness/length/section-count is explicitly **not** a target. (2) Venue-fit is checked
against a concrete **acmart-sigconf + ICAIF-CFP checklist** (8-page limit, anonymization,
required-section coverage, ACM reference style), not a craft exemplar's vibe. (3) A genuine
open-access ICAIF/finance-ML venue exemplar is obtained and distilled (paraphrase only) to anchor
venue-fit. (4) STOP is redefined objectively: **gate passes + every number bound to the ledger +
three evidence domains separate + a fresh review finds nothing Critical/Important + Codex finds no
overclaim.** "Exemplar moves present" is removed from the stop gate.

**Consequences:** "Done" is now fully checkable. Exemplar-conformance can no longer block
completion or drag the prose toward the wrong genre; it informs drafting only.
