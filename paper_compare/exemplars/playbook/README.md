8-page ICAIF protocol paper (contribution = a guarded evaluation protocol on a near-null

## Sources distilled
- **DLinear** = Zeng, Chen, Zhang & Xu, *Are Transformers Effective for Time Series
  Forecasting?* (AAAI 2023 / arXiv 2205.13504). The "simple baseline beats complex models,
  sold through rigor" paper.
- **Grinsztajn** = Grinsztajn, Oyallon & Varoquaux, *Why do tree-based models still
  outperform deep learning on tabular data?* (NeurIPS 2022 D&B / arXiv 2207.08815). The
  "benchmark/methodology IS the contribution" paper.

## HARD reminders (read before using any move)
- **Paraphrase only.** Every illustration here is reworded; never lift an exemplar sentence.
- **Red-line verbs override structure.** Both exemplars use *outperform / surprisingly
  outperforms / state-of-the-art / best / large margin*. Our red lines FORBID
  `outperforms / best / significant`. **Borrow the move, swap the verb** for ledger-bound
  red-line vocabulary ("exceeds the same-row dummy floor", "meets the predeclared criteria",
  "+1.69pp over the ~0.50 dummy"). Files flag this inline with **[RED-LINE]**.
- These are deflationary papers; keep only the honest-rigor craft that fits a near-null case.
